# VoicePicker

配音員試音檔審閱工具。在一整包音檔上記 timecode 評語、掛角色標籤、打分數，用角色 Dashboard 快速整理，匯出評語報告或分享試聽版給同事。

**v0.8.0 · 純前端 · 無後端 · 無帳號**

---

## 使用方式

1. 用 **Chrome 或 Edge** 開啟（需要 File System Access API）
2. 點「選擇資料夾」，選取 NAS 上的音檔目錄
3. 開始試聽、留言

---

## 三種視圖

| 視圖 | 切換方式 | 用途 |
|------|----------|------|
| 單檔播放器 | 預設 / Tab 從 Grid 返回 | 試聽當前音檔，記 timecode 留言 |
| Grid 網格 | Tab | 一眼瀏覽所有音檔與留言數，快速跳轉 |
| 角色 Dashboard | 右欄頂部按鈕 | 跨音檔依角色彙整所有留言，依分數排序 |

---

## 鍵盤快捷鍵

### 單檔模式

| 按鍵 | 動作 |
|------|------|
| `Space` | 播放 / 暫停 |
| `A` / `←` | 後退 5 秒 |
| `D` / `→` | 前進 5 秒 |
| `W` / `↑` | 跳至上一個留言點 |
| `S` / `↓` | 跳至下一個留言點 |
| `C` / `X` | 在當前秒數開啟留言框 |
| `Tab` | 切換到 Grid 視圖 |

### Grid 視圖

| 按鍵 | 動作 |
|------|------|
| `W/A/S/D` / 方向鍵 | 移動選取 |
| `Space` / `Enter` | 開啟選取的音檔 |
| `Tab` / `Esc` | 返回單檔 |

### 角色 Dashboard

| 按鍵 | 動作 |
|------|------|
| `W` / `↑` | 移到上一張留言卡 |
| `S` / `↓` | 移到下一張留言卡 |
| `Space` | 就地播放 / 暫停（留在 Dashboard）|
| `Enter` | 進單檔模式，跳至該留言的 timecode |
| `Shift + 1–5` | 為當前聚焦的留言卡打分 |
| `Tab` / `Esc` | 返回單檔 |

### 留言框（Composer）

| 按鍵 | 動作 |
|------|------|
| `Enter` | 送出留言 |
| `Shift + Enter` | 換行 |
| `↓` / `Tab` | 游標移至「對應秒數」切換鈕 |
| `↓` 繼續 | 進入角色選取區 |
| `↑` | 往上返回 |
| `Space` | 切換秒數模式，或選取/取消角色 |
| `Shift + 1–5` | 設定評分（任何焦點位置均有效）|
| `Esc` | 取消並關閉 |

## 功能

- **Timecode 留言**：記在波形上，可點擊跳轉；也可選「整體留言」不綁定秒數
- **角色標籤**：主角 / 配角 / 吉祥物三類，可複選；支援新增、刪除、雙擊重命名
- **1–5 星評分**：可在 Composer 輸入或在留言卡上直接按
- **角色 Dashboard**：跨音檔依角色彙整，分數高到低排列
- **協作同步**：選定資料夾後每 30 秒輪詢 `voicepicker.json`，自動合併隊友新增的留言（僅新增，不覆蓋本地）
- **評語報告 PDF**：依角色分組，含 timecode、音檔名、評語、評分、留言者
- **唯讀分享版**：匯出自包含 HTML，同事選同一個 NAS 資料夾即可試聽，無法編輯留言
- **備份匯出 / 匯入**：JSON 格式，可換電腦或留存記錄

---

## 協作使用

多人同時審閱同一批音檔：

1. 大家用 Chrome/Edge 開啟同一個 VoicePicker 網址
2. 各自選取 NAS 上**相同的音檔資料夾**
3. 留言後自動寫入 `voicepicker.json`，每 30 秒合併同步
4. 審閱完成後，匯出「唯讀分享版」HTML 給導演或外包

> **注意**：同一秒同時送出留言有極低機率衝突，下一輪同步會補齊。適合非同步審閱流程（一人在聽另一人先聽完），不適合 Google Docs 等級的即時協同。
